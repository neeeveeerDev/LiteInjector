# ⚡ liteInjector:
Лёгкий, быстрый и безопасный DLL-инжектор для Windows, написанный на чистом C++17. Не требует установки, не пишет в системный реестр и не оставляет следов. Идеально подходит для внедрения модов, дэбаггеров и кастомных библиотек в игры и приложения.


🚀 Особенности
Консольный интерфейс: Управление в два клика, ничего лишнего.
Список процессов: Вывод всех запущенных PID и имён процессов.
Drag & Drop: Просто перетащи .dll файл в консоль, путь пропишется сам.
Минимальные зависимости: Нужен только user32.lib.
Статическая линковка: Один .exe файл, который запустится на любой Windows 10/11.

📖 Как использовать
Запусти liteInjector.exe от имени администратора (чтобы получить права на чтение памяти других процессов).
Нажми 1, чтобы вывести список всех запущенных процессов. Найди нужный PID.
Нажми 2, чтобы начать инжект.
Перетащи .dll файл в консоль (или введи путь вручную) и нажми Enter.
Введи PID целевого процесса и нажми Enter.
Если всё прошло успешно, ты увидишь сообщение [+] Successfully injected!.
